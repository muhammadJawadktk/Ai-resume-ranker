# Resume Ranker (BERT-Based Candidate Matching)

Automated resume screening system that ranks candidates against job descriptions using a BERT-based transformer model.

## Problem
Manual resume screening is slow and error-prone. This project automates matching so HR teams can quickly identify the most suitable candidates.

## Approach
- Encodes resumes and job descriptions using BERT embeddings
- Scores and ranks candidates by semantic similarity to the job description
- Outputs a ranked shortlist instead of raw keyword matches

## Result
~92% accuracy in matching candidates to relevant job postings, cutting manual screening time significantly.

## Stack
Python, BERT (Transformers), scikit-learn
